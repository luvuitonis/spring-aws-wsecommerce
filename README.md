# E-Commeerce 📊

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Type](https://img.shields.io/badge/artifact-JAR-orange.svg)

Proyecto para la evaluación técnica de Microservicios.
 Diseñado para un desarrollador Spring Boot Microservicios.
---

## 🚀 Características Principales

* **Ejercicio práctico:** Evaluar  habilidades en Spring Boot, JPA, pruebas unitarias y buenas prácticas.
* **Pruebas Unitarias y de Integración:** Pruebas Unitarias (Service Layer) y Pruebas Unitarias (Controller Layer).

---

## 🛠️ Requisitos Previos

Antes de instalar y ejecutar el proyecto, asegúrate de contar con el siguiente software en tu equipo:

* **Java Development Kit (JDK):** Versión 11 o superior.
* **Spring Boot:** 2.7+ o 3.x.
* **Gestor de Dependencias:** Maven 3.5+.
* **Base de Datos:** H2.
* **Integración de pruebas:** JUnit 5 + Mockito.
* **Mapeo entre beans:** MapStruct (opcional)
* **Procesador de anotaciones:** Lombok

---

## 📦 Instalación y Empaquetado

Sigue estos pasos para compilar, generar el artefacto y poner en marcha la aplicación:

### 1. Clonar el repositorio
```bash
git clone [https://github.com/tu-usuario/estadistica-cuantica.git](https://github.com/tu-usuario/estadistica-cuantica.git)
cd ecommerce/target
java -jar ecommerce-1.0.0.jar --spring.profiles.active=dev
```
